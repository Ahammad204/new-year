# New Year - Goals & Resolutions Platform

## 📋 Project Overview

**New Year** is a personal portfolio-style website that combines reflection and goal-setting. It presents a compelling narrative of past challenges and future aspirations, designed to inspire both the owner and visitors.
A modern, motivational website showcasing personal goals, reflections, and new year resolutions with a focus on positive mindset and self-improvement.

## 🔗 Live Demo

Visit the live site: **[https://ahammad204.github.io/new-year/](https://ahammad204.github.io/new-year/)**

### Key Sections

1. **Hero Section** - Motivational header with the tagline "All Things Are Possible If You Believe"
2. **Reflections** - Honest assessment of past failures and lessons learned from the previous year
3. **My Plan** - Four core objectives for the new year:
   - 💻 Web Development - Starting web development journey
   - 🔧 JavaScript - Mastering JavaScript programming
   - 🚭 Smoking Cessation - Quitting smoking habit
   - 🛒 E-Commerce Development - Building e-commerce projects

4. **Call-to-Action** - Positive mindset reinforcement with contact options
5. **Footer** - Contact information and social media links

## 🎨 Design & Features

- **Responsive Design** - Mobile-friendly layout with viewport optimization
- **Modern Styling** - Clean, professional CSS with flexbox layouts
- **Custom Typography** - Google Fonts (Open Sans) for enhanced readability
- **Visual Assets** - Custom banner images, gradients, and icons
- **Social Integration** - Links to Facebook, Twitter, and YouTube
- **Newsletter Signup** - Email subscription feature for audience engagement

## 🚀 Getting Started

### Local Development

1. Clone the repository:
```bash
git clone https://github.com/ahammad204/new-year.git
cd new-year
```

2. Open in your browser:
   - Simply open `index.html` in your preferred web browser
   - No build tools or server setup required

### File Customization

- **index.html** - Edit content, headings, and descriptions
- **style/style.css** - Modify colors, spacing, and typography
- **images/** - Replace images with your own assets

## 📝 Content Areas to Customize

1. **Hero Section** - Update the main heading and description
2. **Failures List** - Replace with your personal reflections
3. **Goals/Habits** - Modify the four main objectives
4. **Contact Details** - Update email and phone number in footer
5. **Social Links** - Add your actual social media profiles
